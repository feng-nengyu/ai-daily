9 arxiv:arxiv-cs-ai

技术方案
FORGE 通过分阶段、基于种群的协议，利用自然语言记忆优化 ReAct 代理。内环使用反思代理将失败轨迹转化为可重用知识，外环在阶段间传播最佳实例记忆并冻结收敛实例。
